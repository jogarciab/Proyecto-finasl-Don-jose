# Sistema de Gestión de Tienda - Flask

Proyecto desarrollado con:
- Python
- Flask
- HTML
- CSS
- JavaScript

El sistema simula una aplicación móvil para la gestión de productos, ventas, facturación, reportes y compras realizadas.
